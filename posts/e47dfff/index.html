使用 nvm 管理 node 版本

最近频繁地被 node 版本伤脑筋…

  • 起因是 slidev，这个库看起来很好玩，事实上确实挺好玩的，但是需要 node 版本>=14
  • 在学习 node.js 的时候，我想安装最新的 v16 版本
  • 需要用 node-mirai 开发 qq 机器人的 canvas 功能的时候，安装 node-canvas 疯狂报错，最后去 issue 里找到了一种降低 node 版本的解决办法…

好烦啊
