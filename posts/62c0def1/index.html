з”ҹе‘Ҫе‘ЁжңҹйҖүйЎ№еҗҚз§°зҡ„еҸҳеҢ–

еңЁ prop зҡ„й»ҳи®ӨеҮҪж•°дёӯи®ҝй—® this

з”ҹжҲҗ prop й»ҳи®ӨеҖјзҡ„е·ҘеҺӮеҮҪж•°дёҚиғҪеҶҚи®ҝй—® this

иҮӘе®ҡд№үжҢҮд»Ө API е·Іжӣҙж”№дёәдёҺз»„д»¶з”ҹе‘Ҫе‘ЁжңҹдёҖиҮҙ

жҢҮд»Өзҡ„й’©еӯҗеҮҪж•°иў«йҮҚе‘ҪеҗҚ

еңЁ 2.x дёӯпјҢиҮӘе®ҡд№үжҢҮд»ӨжҳҜйҖҡиҝҮдҪҝз”ЁйһӢйқўеҲ—еҮәзҡ„зӢ—еӯҗжқҘеҲӣе»әзҡ„пјҢиҝҷдәӣй’©еӯҗжҳҜеҸҜйҖүзҡ„

  • bind - жҢҮд»Өз»‘е®ҡеҲ°е…ғзҙ еҗҺеҸ‘з”ҹпјҢеҸӘеҸ‘з”ҹдёҖж¬Ў
  • inserted - е…ғзҙ жҸ’е…ҘзҲ¶ DOM еҗҺеҸ‘з”ҹ
  • update - еҚ•е…ғж•°жӣҙж–°пјҢдҪҶеӯҗе…ғзҙ е°ҡжңӘжӣҙж–°ж—¶пјҢе°Ҷи°ғз”ЁжӯӨй’©еӯҗ
  • componentUpdated - дёҖж—Ұз»„д»¶е’Ңеӯҗзә§иў«жӣҙж–°пјҢе°ұдјҡи°ғз”ЁжӯӨй’©еӯҗ
  • unbind - жҢҮд»Өиў«з§»йҷӨж—¶и°ғз”ЁпјҢеҸӘи°ғз”ЁдёҖж¬Ў

еңЁ 3.x дёӯпјҢиҮӘе®ҡд№үжҢҮд»ӨжңүдәҶж–°зҡ„ APIпјҢзҺ°еңЁе°Ҷз»„д»¶з”ҹе‘Ҫе‘Ёжңҹе’ҢиҮӘе®ҡд№үжҢҮд»Өзҡ„з”ҹе‘Ҫе‘Ёжңҹз»ҹдёҖдәҶиө·жқҘ

  • created - ж–°зҡ„пјҒеңЁе…ғзҙ зҡ„ attribute жҲ–дәӢд»¶дҫҰеҗ¬еҷЁеә”з”Ёд№ӢеүҚи°ғз”ЁгҖӮ
  • bind вҶ’ beforeMount
  • inserted вҶ’ mounted
  • beforeUpdateпјҡж–°зҡ„пјҒиҝҷжҳҜеңЁе…ғзҙ жң¬иә«жӣҙж–°д№ӢеүҚи°ғз”Ёзҡ„пјҢеҫҲеғҸз»„д»¶з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗгҖӮ
  • update вҶ’ з§»йҷӨпјҒжңүеӨӘеӨҡзҡ„зӣёдјјд№ӢеӨ„иҰҒжӣҙж–°пјҢжүҖд»ҘиҝҷжҳҜеӨҡдҪҷзҡ„пјҢиҜ·ж”№з”Ё updated
  • componentUpdated вҶ’ updated
  • beforeUnmountпјҡж–°зҡ„пјҒдёҺз»„д»¶з”ҹе‘Ҫе‘Ёжңҹй’©еӯҗзұ»дјјпјҢе®ғе°ҶеңЁеҚёиҪҪе…ғзҙ д№ӢеүҚи°ғз”ЁгҖӮ
  • unbind -> unmounted

жІЎжңүзү№ж®ҠжҢҮд»Өзҡ„ж Үи®° (v-if/else-if/elseгҖҒv-for жҲ– v-slot),е°Ҷдјҡз”ҹжҲҗеҺҹз”ҹ<template>е…ғзҙ пјҢиҖҢдёҚжҳҜжёІжҹ“еҶ…йғЁеҶ…е®№

vue3 дёӯпјҢе®№еҷЁжң¬иә«дёҚеҶҚжҳҜжЁЎжқҝзҡ„дёҖйғЁеҲҶ

еңЁ Vue 2.x дёӯпјҢеә”з”Ёж №е®№еҷЁзҡ„ outerHTML е°ҶжӣҝжҚўдёәж №з»„д»¶жЁЎжқҝ (еҰӮжһңж №з»„д»¶жІЎжңүжЁЎжқҝ/жёІжҹ“йҖүйЎ№пјҢеҲҷжңҖз»Ҳзј–иҜ‘дёәжЁЎжқҝ)гҖӮVue 3.x зҺ°еңЁдҪҝз”Ёеә”з”Ёе®№еҷЁзҡ„ innerHTMLпјҢиҝҷж„Ҹе‘ізқҖе®№еҷЁжң¬иә«дёҚеҶҚиў«и§ҶдёәжЁЎжқҝзҡ„дёҖйғЁеҲҶгҖӮ

attribute зҡ„ејәеҲ¶иЎҢдёә

иҝҷжҳҜдёҖдёӘдҪҺзә§зҡ„еҶ…йғЁ API жӣҙж”№пјҢдёҚдјҡеҪұе“ҚеӨ§еӨҡж•°ејҖеҸ‘дәәе‘ҳгҖӮ

  • еҲ йҷӨжһҡдёҫ attribute зҡ„еҶ…йғЁжҰӮеҝөпјҢе№¶е°Ҷиҝҷдәӣ attribute и§Ҷдёәжҷ®йҖҡзҡ„йқһеёғе°” attribute
  • йҮҚеӨ§ж”№еҸҳпјҡеҰӮжһңеҖјдёәеёғе°”еҖјпјҢеҲҷдёҚеҶҚеҲ йҷӨ attribute falseгҖӮзӣёеҸҚпјҢе®ғиў«и®ҫзҪ®дёә attr=вҖңfalseвҖқгҖӮз§»йҷӨ attributeпјҢдҪҝз”Ё null жҲ–иҖ… undefinedгҖӮ

еңЁ 2.xпјҢжҲ‘д»¬жңүд»ҘдёӢзӯ–з•ҘжқҘејәеҲ¶ v-bind зҡ„еҖјпјҡ

  • еҜ№дәҺжҹҗдәӣ attribute/е…ғзҙ еҜ№пјҢVue е§Ӣз»ҲдҪҝз”Ёзӣёеә”зҡ„ IDL attribute(property)пјҡжҜ”еҰӮ value зҡ„ <input>пјҢ<select>пјҢ<progress>пјҢзӯүзӯү
  • еҜ№дәҺвҖңжһҡдёҫ attributeвҖқ (зӣ®еүҚ contenteditableпјҢdraggable е’Ң spellcheck)пјҢVue дјҡе°қиҜ•ејәеҲ¶е°Ҷе®ғд»¬дёІиө·жқҘ (зӣ®еүҚеҜ№ contenteditable еҒҡдәҶзү№ж®ҠеӨ„зҗҶпјҢдҝ®еӨҚ vuejs/vue#9397)
  • еҜ№дәҺе…¶д»– attributeпјҢжҲ‘д»¬з§»йҷӨдәҶ falsy еҖј (undefinedпјҢnullпјҢor false) е№¶жҢүеҺҹж ·и®ҫзҪ®е…¶д»–еҖј (и§ҒиҝҷйҮҢ)гҖӮ

еңЁ vue3 дёӯпјҢж”ҫејғвҖңжһҡдёҫ attributeвҖқзҡ„еҶ…йғЁжҰӮеҝөпјҢе№¶е°Ҷе®ғд»¬и§Ҷдёәжҷ®йҖҡзҡ„йқһеёғе°” HTML attributeгҖӮ

  • иҝҷи§ЈеҶідәҶжҷ®йҖҡйқһеёғе°” attribute е’ҢвҖңжһҡдёҫ attributeвҖқд№Ӣй—ҙзҡ„дёҚдёҖиҮҙжҖ§
  • е®ғиҝҳеҸҜд»ҘдҪҝз”Ё вҖҳtrueвҖҷ е’Ң вҖҳfalseвҖҷ д»ҘеӨ–зҡ„еҖјпјҢз”ҡиҮіеҸҜд»ҘдҪҝз”Ё contenteditable зӯү attribute зҡ„е…ій”®еӯ—`

  • иҝҷи§ЈеҶідәҶ true е’Ң false д№Ӣй—ҙзҡ„дёҚдёҖиҮҙжҖ§пјҢе№¶дҪҝиҫ“еҮә aria-* attributes жӣҙе®№жҳ“
